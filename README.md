# Supporto Rewindly

Benvenuto/a nella pagina di supporto di Rewindly. Qui puoi trovare risposte alle domande frequenti e una mail di contatto per ulteriore assistenza.

## FAQ

**Q:** Per quali piattaforme è disponibile l’app?  
**A:** L'app è attualmente disponibile per iPhone, iPad e Mac.

**Q:** Quanto costa l'applicazione?  
**A:** Rewindly è totalmente gratuita e priva di pubblicità.

**Q:** Quali dati vengono raccolti?  
**A:** L'app non raccoglie alcun dato. Accede ai brani della libreria musicale dell'utente ne mostra le statistiche. Nessun dato viene memorizzato al di fuori dell'app.
